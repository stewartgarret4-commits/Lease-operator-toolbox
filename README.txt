Lease Operator Toolbox

Included calculators:
- Well test calculator
- Gauge modes:
  - Feet and inches, such as 3' 6"
  - Decimal feet / feet and tenths, such as 3.50
  - Total inches
- Water and gas meter inputs
- ESP dynamic fluid-level estimate
- Basic oilfield conversions
- Offline support after installation

To install on iPhone:
1. Host these files on any HTTPS website.
2. Open index.html through the hosted website in Safari.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open the new Lease Toolbox icon.

Important:
Apple does not allow a Home Screen web app to be installed directly from a ZIP file or the Files app.
The files must first be placed on a secure website. Free options include GitHub Pages, Netlify, or Cloudflare Pages.
