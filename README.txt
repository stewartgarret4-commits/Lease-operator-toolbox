GITHUB PAGES UPDATE
1. Extract this ZIP.
2. In your GitHub repository choose Add file > Upload files.
3. Upload all extracted files and replace the old files.
4. Commit changes and wait a few minutes.
5. Open the GitHub Pages address in Safari.
6. Tap Share > Add to Home Screen.

This is a field-test prototype. Data is stored locally in Safari on that device. Clearing website data can erase it.
