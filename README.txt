Lease Operator Toolbox v2

This version supports 1, 2, or 3 oil tanks. Each tank has its own start gauge, end gauge, and editable bbl-per-inch factor. Oil from all selected tanks is totaled automatically.

To update GitHub Pages:
1. Extract this ZIP.
2. In your repository, choose Add file > Upload files.
3. Upload all extracted files and replace the old versions.
4. Commit changes.
5. Reopen the Home Screen app after GitHub Pages updates.
